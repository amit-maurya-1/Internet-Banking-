## 📌 Overview

The **Indian Internet Banking Portal** is a fully functional, multi-page frontend banking application that replicates the look, feel, and features of real Indian bank net banking portals. Built entirely with  HTML, CSS, and JavaScript.

It includes everything a real banking portal would have: secure login with CAPTCHA, fund transfers (NEFT/IMPS/RTGS), bill payments, card management, fixed deposit tools, and a complete profile & security system.

> ⚡ **Just open `index.html` in your browser — that's it.**

---

## ✨ Features

### 🔐 Authentication
- Customer ID + Password login with form validation
- Dynamic CAPTCHA generation with refresh
- Forgot Password modal flow
- 
### 🏠 Dashboard
- Live account balance cards (Savings + Current) with show/hide toggle
- Summary stat cards: salary, expenses, FD value, transaction count
- 8 quick-action shortcut tiles
- Recent transactions panel

### 💸 Fund Transfer
- Transfer type tabs: **NEFT**, **IMPS**, **RTGS**, **Own Account**
- Saved beneficiary dropdown with autofill
- OTP verification modal with 30-second countdown timer
- Printable transfer receipt with reference number

## 🗂️ Project Structure

```
indian-banking-portal/
│
├── index.html          # Login & authentication page
├── dashboard.html      # Main dashboard with charts & quick actions
├── transfer.html       # Fund transfer — NEFT / IMPS / RTGS
├── transactions.html   # Transaction history, filters, CSV export
├── bills.html          # Bill payments & recharge
├── cards.html          # Debit & credit card management
├── deposits.html       # Fixed deposits & cheque services
├── profile.html        # Profile, security & alert settings
│
├── style.css           # Global design system (variables, components, layout)
└── script.js           # Shared utilities, demo data, session helpers
```
